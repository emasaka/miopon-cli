# miopon-cli

CLI client of IIJmio coupon switch API

## SYNOPSIS

    miopon [subcommand]

## DESCRIPTION

Manipulates IIJmio mobile SIM's coupon information.

## SUBCOMMANDS

* info
* log
* on 
* off

(TBD)

## REQUIREMENTS

* Ruby (>= 1.9)
* PhantomJS (It's not necessary if you have active access token)
* developer ID of IIJmio coupon switch API
